## Install

```sh
npm install
```

## Usage
1. Start the server ```npm run start```
2. Set the SENDIT_API ```export SENDIT_API=http://127.0.0.1```
3. Upload a file ```./sendit.sh FILE```
4. Download a file ```./getit.sh ID```
